<p>Android OS contains a default Keyboard and KeyboardView implementations (deprecated as of Android 10, but still available). Input method developers can use these classes as base for their own keyboard implementations. <i>Tiny Keyboard</i> is an implementation without any changes.</p></ul><li>Smallest possible APK size of 22kB (as of version 0.3)</li><li>Permissions: 0</li><li>Features: it has keys</li><li>Supported layouts: en_US</li><li>No Launcher icon and Settings</li></ul><p><b>To enable the keyboard:</b></p><ol><li>Go to: System Settings - Language and Input - Virtual Keyboard - Manage Keyboards (path may be different across Android versions and brands).</li><li>Enable the Tiny Keyboard (you my get a default warning about tracking, don't worry, it can't track what you type due to the lack of Network permission)</li><li>Switch to Tiny Keyboard from current Input Method (differs between keyboards, usually long-press space)</li></ol>
